# willhaben-tui

A React-based Terminal User Interface for browsing willhaben.at.

## Features

- Search willhaben.at directly from the terminal
- Uses your browser's cookies for authentication
- Built with Ink

## Getting Started

```bash
npm install
npm start
```

## Requirements

- Node.js 18+
- A browser with willhaben.at session cookies (Chrome, Edge, Firefox, or Safari)
